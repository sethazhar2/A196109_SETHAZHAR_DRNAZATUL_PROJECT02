## Project Overview

HydroTrack is a mobile application developed using Kotlin and Jetpack Compose to help users monitor their daily water intake and maintain healthy hydration habits.

This project supports **SDG 6: Clean Water and Sanitation** by encouraging users to track water consumption and improve awareness of hydration.

---

## Features

### Home Screen
- Track daily water intake
- View hydration progress
- Quick water logging

### Stats Screen
- Display hydration statistics
- Daily goal monitoring

### Profile Screen
- User profile information
- Daily hydration goal

### Summary Screen
- View saved hydration records
- Update and delete logs

### Daily Tips Screen
- Hydration advice and reminders

### Weather Screen
- Live temperature data
- Wind speed information
- Open-Meteo API integration

### Location Screen
- GPS location detection
- Displays latitude, longitude, and location name

### Community Screen
- Upload hydration tips
- Firebase Firestore integration
